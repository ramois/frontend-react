# Proyecto React - Diplomado

Guía básica para levantar el proyecto en local.

**Requisitos**
- Node.js 18+ (recomendado 20+)
- npm 9+

**Instalación**
```bash
npm install
```

**Variables de entorno**
El proyecto usa `VITE_API_URL` para apuntar al backend.

Archivo de ejemplo: `.env.sample`

Ejemplo:
```bash
VITE_API_URL=https://carlos-trigo.onrender.com/api
```

**Levantar en desarrollo**
```bash
npm run dev
```

**Build de producción**
```bash
npm run build
```

**Preview del build**
```bash
npm run preview
```
